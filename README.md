# Fyntrum Community Dashboard

Plataforma privada de la comunidad Fyntrum — trading algorítmico con FT01 + MBS v8.0.

## Stack
- HTML + CSS + JS puro
- Supabase (autenticación + base de datos)
- Google Sheets (clases, eventos, descargables, config)
- Vercel (hosting)

## Despliegue
1. Importa este repositorio en Vercel
2. Deploy automático — sin configuración adicional
